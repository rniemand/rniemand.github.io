## ESP8266

### LoLin NodeMCU V3
![](/assets/img/2017/2017-04-05/001.jpg)

```
SD CARD
VCC ............ 3.3v
GND ............ GND
MISO ........... D6
MOSI ........... D7
SCK ............ D5
CS ............. D8
```

### WeMos ESP8266 D1 Mini
![](/assets/img/2017/2017-04-05/002.jpg)

```
I2C
  SDA => D2
  SCL => D1

SPI
  M-CLK => D5
  MISO => D6
  MOSI => D7
```
