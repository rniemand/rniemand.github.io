This is a quick reference post on updating your local Pi-hole installation, the process can be completed in 3 simple steps:

## SSH into your Pi

```shell
SSH into your Pi
```

## Check for updates
```shell
pihole -v
```

### Run the update
```shell
pihole -up
```
